## Design Pattern

Design pattern is well described reusable solution to common problem in
software design and development.

Patterns aren't invented, they are discovered.

### Creational

- Factory Method
- Simple Factory
- Abstract Factory
- Parameterized Factory
- Builder
- Prototype
- Singleton
- Object pool

### Structural

- Decorator : enhance
- Adaptor
- Facade
- Proxy : control
- Composite
- Flyweight

### Behavioral

- Strategy
- Observer (EventPropagator: EventListener)
- Delegation
- Chain of responsibility
- Iterator
- Null-Empty Object
- State
